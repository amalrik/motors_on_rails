# Teste Ruby para Candidatos

## Setup

1. Clone do repo
2. bundle install
3. crie o database.yml(deixei um sample basta renomear).
4. bundle exec rake db:create db:migrate db:seed
5. bundle exec rake webmotors:update_makers webmotors:update_models
6. rails s
